# Barvon - Web Frontend

The web interface for Barvon, a 2D MMO dragon game. Manage your account, trade items, and customize your profile.

## Features

- **Item Trading**: Browse and trade items with other players through an offer system
- **Inventory Management**: View your items and create sell offers
- **Profile Customization**: Upload and manage your profile picture
- **Account Management**: 
  - User registration with email confirmation
  - Password reset and change
  - Account deletion
  - Secure authentication with access and refresh tokens
- **Balance Tracking**: Monitor your in-game currency

## Tech Stack

- React 19.2 with TypeScript
- React Router for navigation
- CSS Modules for styling
- Context API for state management
